# How to Add Professional Images to Your Site

## Step 1: Create Image Folders

Run these commands in your terminal:

```bash
mkdir -p public/images/cases
mkdir -p public/images/team
mkdir -p public/images/insights
```

## Step 2: Download Free Professional Images

### For Homepage Hero Image
1. Go to: https://unsplash.com/s/photos/business-consulting
2. Look for images with:
   - Professional office setting
   - Clean, modern aesthetic
   - Neutral colors (white, gray, blue tones)
   - Team collaboration or strategy planning
3. Download a high-quality image (1200x900px or larger)
4. Save as: `public/images/hero-consulting.jpg`

**Recommended searches on Unsplash:**
- "business strategy meeting"
- "professional office workspace"
- "corporate team collaboration"
- "modern office interior"

### For Featured Case Study
1. Go to: https://unsplash.com/s/photos/project-management
2. Look for:
   - Dashboard or analytics screens
   - Project planning boards
   - Professional technology workspace
3. Download and save as: `public/images/case-study-erp.jpg`

### For Individual Case Studies
Download 4 images for each case study:

1. **ERP Programme** - Search: "enterprise software dashboard"
   - Save as: `public/images/cases/erp-programme.jpg`

2. **Go-Live Readiness** - Search: "product launch preparation"
   - Save as: `public/images/cases/golive-readiness.jpg`

3. **Transformation** - Search: "digital transformation business"
   - Save as: `public/images/cases/transformation.jpg`

4. **Website Delivery** - Search: "web design development"
   - Save as: `public/images/cases/website-delivery.jpg`

## Step 3: Optimize Images (Optional but Recommended)

Use online tools to compress images:
- **TinyPNG**: https://tinypng.com/
- **Squoosh**: https://squoosh.app/

Target file size: Under 200KB per image

## Step 4: Activate Images in Code

Once images are in place, the code will automatically display them. The homepage is already set up!

## Image Selection Tips

✅ **DO:**
- Choose clean, professional images
- Use neutral color palettes
- Select high-resolution photos
- Ensure images are free for commercial use
- Pick images that match your brand tone

❌ **DON'T:**
- Use overly staged stock photos
- Choose images with bright, clashing colors
- Download low-resolution images
- Use images with visible watermarks
- Select images that don't relate to your content

## Specific Recommendations

### Hero Image Style
Look for images similar to McKinsey's aesthetic:
- Clean, minimal composition
- Professional but not stuffy
- Neutral backgrounds
- Natural lighting
- Authentic moments (not overly posed)

### Color Palette to Match
Your site uses:
- Teal: #2F5D62
- White: #FFFFFF
- Light Gray: #F8F9FA
- Dark Gray: #6B7280

Choose images with similar tones or neutral colors that won't clash.

## After Adding Images

The homepage hero section will automatically display your image once you add:
- `public/images/hero-consulting.jpg`

The featured case study will show your image once you add:
- `public/images/case-study-erp.jpg`

No code changes needed - it's already set up!
